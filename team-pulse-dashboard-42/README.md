This repository contains Team Pulse, a productivity dashboard for tracking team member status and managing tasks.

## Quickstart (local development)

Requirements: Node.js (16+), npm

1. Install dependencies

```sh
npm install
```

2. Start the dev server

```sh
npm run dev
```

3. Open your browser at http://localhost:8080

## Editing the project

You can edit the source files in `src/`. The app is a React + TypeScript project using Vite and Tailwind.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Build for production:

```sh
npm run build
```

Then serve the `dist/` directory with your preferred hosting provider.
